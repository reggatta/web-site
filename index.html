About Us

Reggatta Systems, Inc. was started back in 1996 with the simple idea of building software to delight people.

We accidentally found ourselves developing code to access Btrieve  databases from Turbo Pascal, which led to the development of the Titan Database Interface For Btrieve, a developer's toolset that allowed lighting fast access. Reggatta Systems developed the world's first Borland (now Embarcadero) Database Engine replacement back in 1995. For over 30 years we have worked closely with the developers of Delphi to bring you the very best database interface products for their development tools. We are proud to be a Embarcadero Technology Partner.

We have been developing code for the iPhone since 1988!  You may not believe that, but we worked on the original NeXT computer and it included an amazing GUI tool called Interface Builder - the very same Interface Builder used in the modern Xcode to develop IOS and Mac Apps.

We are here to help people do great things with technology and have some fun at the same time. If there is something you need, please reach out.

Titan Database Interface for Btrieve

Back in 1996 we needed to get at some data stored in a Btrieve database and there were not good tools out there to do that. We wrote our own tools that were lightning fast and developers found out about them.

What is Titan?

The Titan Database Interface is a software developer's tool which provides highly-optimized database access components for the latest Embarcadero Rad Studio Delphi™ XE verions and C++ Builder™ development environments. (We have versions that go back to Delphi 1.0, but nobody needs those anymore)

How does it work?

Titan provides a set of components similar to the standard TTable and TQuery components, that talk directly to the Btrieve database engine. As Btrieve developers ourselves, we wanted the awesome combination of the design-time access to data, as well as lightning fast access to Btrieve. No ODBC could get in our way! Titan TtbTable and TtbQuery components are compatible with any other components that work with data. This means you can use your InfoPower, DevExpress, or any other data-aware components with Titan.

Are you still publishing Titan?

With the advent of the virtual Dataset (TDataSet) in the more recent versions of Delphi, the Titan Database components were no longer needed. We have discontinued new releases. We continue to hear from some developers who are still using older versions of Titan and are happy to help as consultants when the need arises.
